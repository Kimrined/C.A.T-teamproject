블로그의 전체 기능

1. 사용자는 회원가입을 진행.
 + 이메일 주소, 비밀번호, 비밀번호 확인, 닉네임, 핸드폰 번호,
주소, 상세 주소를 입력해서 회원가입을 진행.

2. 회원가입이 성공적으로 이루어지면 로그인 진행.
 + 로그인 이메일과 패스워드로 로그인을 진행.

3. 로그인에 성공하면 메인 화면으로 이동.
 + 메인화면에는 주간 Top3 게시글이 보임.
 + 주간 Top3의 경우, 좋아요 순으로 정렬.
 + 최신 게시물이 리스트 형식으로 출력.
 + 한 페이지에 5개씩 게시물을 보임.
 + 페이징 처리가 되며 최대 10 페이지까지 보임.
 + 이전 섹션과 이후 섹션으로 이동하는 방향 버튼 有

4. 인기 검색어가 우측에 존재.

5. 상단에는 네비게이션 바
 + 네비게이션 바에서 검색을 할 수 있음
 + 네비게이션 바에서 마이페이지로 이동 가능

6. 마이페이지로 이동.
 + 프로필 사진 수정 및 닉네임 수정
 + 마이페이지에서 게시물 등록

7. 게시물
 + 게시글 등록 : 제목, 내용, 사진, 동영상, 파일
 + 본인 게시물이라면 게시물을 수정.
 + 게시물 수정 : 제목, 내용, 사진, 동영상, 파일
 + 본인 게시물 삭제 가능
 + 본인 게시물 아니면 좋아요 클릭
좋아요 : 유저의 프로필 사진, 닉네임 출력
 + 모든 게시물 댓글 작성 (내용만)
 + 댓글 : 작성자, 작성자 프로필 사진, 작성일, 내용 출력 /
최대 3개만 출력, 페이징 최대 10페이지. (이전 다음 버튼 존재)
 + 댓글, 좋아요 수가 게시물 상세에서도 표시

8. 게시물 리스트
 + 게시물 선택 : 제목, 작성자, 작성자 프로필 사진, 작성일, 내용, 사진, 동영상, 파일 출력.
 + 작성자 프로필 사진, 닉네임, 게시물의 제목, 내용, 좋아요 수, 댓글 수, 조회 수 출력.
